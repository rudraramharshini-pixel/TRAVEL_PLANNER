# 🌍 Travel Planner Website ✈️

![Travel Banner](https://images.unsplash.com/photo-1507525428034-b723cf961d3e)


## 📌 Project Overview

The **Travel Planner Website** is an interactive web application that helps users plan their trips بسهولة (easily).
Users can enter a destination manually or use 🎤 **voice input** for a faster experience.


## ✨ Features

* 🎤 Voice-based destination input
* 🌍 Beautiful dynamic background (tourist places)
* 🧭 Simple and user-friendly interface
* 📋 Trip planning form
* 🌐 Live deployed website


## 🛠️ Technologies Used

* 💻 HTML
* 🎨 CSS
* ⚙️ JavaScript
* 🚀 GitHub Pages (Deployment)

## 🧑‍💻 How It Works

1. Enter your destination in the input field
2. OR click the 🎤 button and speak
3. The system captures your input
4. Plan your trip easily
